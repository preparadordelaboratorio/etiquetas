Guia completa: Generador de Etiquetas SGA/GHS

Esta herramienta te permite crear etiquetas de seguridad para los productos quimicos de tu laboratorio, siguiendo el Sistema Globalmente Armonizado (SGA/GHS), que es el estandar internacional obligatorio para la comunicacion de peligros quimicos.

Tip: Toda la informacion que necesitas para llenar la etiqueta esta en la Ficha de Datos de Seguridad (FDS o SDS) del producto quimico. Si no la tenes, busca en internet el nombre del producto + "ficha de seguridad" o "SDS".

Paso a paso

1 Institucion

Escribi el nombre de tu colegio o laboratorio. Aparece como encabezado en la parte superior de la etiqueta.

2 Identificacion del Producto

Nombre: El nombre comercial o cientifico del producto (ej: "Acido Clorhidrico", "Etanol").

Formula quimica: Escribila normalmente y se convierte sola. Ejemplos:

H2SO4 → H₂SO₄
NaOH → NaOH
Ca(OH)2 → Ca(OH)₂
Fe^3+ → Fe³⁺ (superindice)
2H2+O2->2H2O → 2H₂+O₂ → 2H₂O

Concentracion: El porcentaje o concentracion (ej: "37%", "0.1 M").

N° CAS: Identificador universal del producto quimico. Se puede desactivar con el interruptor si no lo necesitas.

3 Palabra de Advertencia

Segun la clasificacion del producto, elegir:

PELIGRO — Para productos con peligros mas graves (ej: toxico, corrosivo fuerte).

ATENCION — Para productos con peligros menores (ej: irritante, nocivo).

Esta informacion esta en la seccion 2 de la FDS.

4 Pictogramas de Peligro

Los pictogramas son los rombos rojos con simbolos negros que indican visualmente el tipo de peligro. Hace clic en los que correspondan (podes seleccionar varios). Los mas comunes en laboratorios escolares:

  • GHS02 (Llama): Productos inflamables como etanol, acetona.
  • GHS05 (Corrosion): Acidos y bases fuertes como HCl, NaOH.
  • GHS07 (Signo !): Irritantes, nocivos. Muy comun en laboratorio.
  • GHS09 (Medio ambiente): Toxico para organismos acuaticos.

5 Frases H (Indicaciones de Peligro)

Las frases H describen los peligros especificos. Hay dos formas de agregarlas:

a) Buscador: Escribi el codigo (H314) o una palabra clave ("quemadura", "toxico") y hace clic en la frase para agregarla.

b) Pegar codigos: Copia los codigos de la FDS y pegalos en el campo "Pegar frases H aqui". El sistema los detecta automaticamente.

Ejemplo de pegado: H314, H331, H335
Tambien funciona: H314+H331 H335
O simplemente: H314 H331 H335

Podes elegir entre mostrar Solo codigo (ej: H314) o Frase completa (ej: H314: Provoca quemaduras graves...).

6 Frases P (Consejos de Prudencia)

Funcionan igual que las frases H. Las frases P indican que hacer para protegerse. Ej: P280 = "Usar guantes", P305+P351+P338 = "Lavar ojos con agua".

7 Proveedor

Datos del fabricante o distribuidor. Si no lo necesitas, desactivalo con el interruptor "En etiqueta" y la etiqueta aprovecha ese espacio.

8 Tamano de Etiqueta

Elegi un tamano predefinido segun tu recipiente o pone medidas personalizadas en milimetros:

  • Vial 40x30: Para tubos de ensayo y viales pequenos.
  • Frasco chico 60x40: Frascos de 50-100 ml.
  • Frasco med. 80x60: Frascos de 250-500 ml (recomendado).
  • Frasco grande 100x70: Frascos de 1 litro.
  • Botella A6 / Bidon A5: Recipientes grandes y bidones.

En etiquetas chicas, las frases se abrevian automaticamente a solo codigos para que entre todo.

Vista previa y zoom

El panel derecho muestra la etiqueta en tiempo real. Usa los botones + - para hacer zoom y el boton ⊑ para ajustar al tamano de la ventana.

Que es el SGA/GHS?

El Sistema Globalmente Armonizado (SGA), conocido internacionalmente como GHS (Globally Harmonized System), es un sistema estandarizado de la ONU para clasificar y comunicar los peligros de los productos quimicos. Es obligatorio en Uruguay y en la mayoria de los paises.

Una etiqueta SGA contiene: nombre del producto, pictogramas de peligro, palabra de advertencia, frases H y P, e informacion del proveedor. Este generador te ayuda a crear esas etiquetas facilmente.

Importante: Este generador es una herramienta educativa de apoyo. Siempre verifica la informacion con la Ficha de Datos de Seguridad (FDS) oficial del producto quimico.
